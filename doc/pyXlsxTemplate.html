pyXlsxTempalte API¶

XlsxFile¶

class pyXlsxTemplate.XlsxFile(filename=None)¶

An Excel 2007 xlsx file that can be modified without losing formatting, formulas, and charts.

Parameters:filename – The name of the xlsx file.
load(filename)¶

Loads an xlsx file.

Parameters:filename – The name of the xlsx file.
loadxml(name)¶

Loads an XML document from the xlsx file archive.

Parameters:name – The name of the archived file (full path in the compressed archive).
Returns:An xml.dom.minidom object.
resetAllFormulas()¶

Removes the values from all of the formulas in the xlsx file so they will be recalculated when the file is opened again.

save(filename)¶

Saves an xlsx file.

Parameters:filename – The name of the file to save to.
savexml(name, xmldoc)¶

Updates the the file in the xlsx file archive with the xml.dom.minicom XML object.

Parameters:
  • name – The name of the archived file (full path in the compressed archive).
  • xmldoc – The xml.dom.minitom object to be saved.
