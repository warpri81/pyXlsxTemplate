Welcome to pyXlsxTemplate's documentation!
==========================================

**pyXlsxTemplate** is a library for making minor modifications to an Excel 2007
*xlsx* file without losing formatting, formulas, and charts.

The source code can be downloaded from github at
`https://github.com/warpri81/pyXlsxTemplate <https://github.com/warpri81/pyXlsxTemplate>`_.
